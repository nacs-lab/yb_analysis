Channels — FPGA1 DDS / TTL / Clock …

Live readback + control of FPGA1 through the molecube ZMQ daemon (tcp://192.168.0.174:7777) — the same daemon libnacs runs sequences on. Every action goes through the daemon, never the devices directly. Changes apply immediately.

DDS channels

Frequency in MHz, amplitude 0–1, phase in degrees. Edit a field and press Set (or Enter). ovr forces a fixed override value (click again to clear); an overridden value shows ▲. Click a Name to rename it (Enter or ✓ to confirm).

—

NI DAC channels — Dev1 (PCIe-6738) …

Live readback of the NI analog-out voltages, read directly from the PCIe-6738's internal AO monitor (Dev1/_aoN_vs_aognd) — independent of the molecube daemon (which only fronts FPGA1 DDS/TTL). Type a voltage under Set (V) and press Enter/Set to drive a channel: a one-off DC write (like setV; the DAC holds it, ±10 V). Readback + writes are paused while a scan is running (the run owns the NI card).

—

TTL channels

Click a channel to toggle its output. Shift-click cycles the override (normal → force-low → force-high). A forced channel is outlined low / high. Click a channel's name to rename it (Enter / ✓ to confirm).

The command-list the daemon runs on boot (read-only).
